## Requirements
```ruby
'CocoaLumberjack/Swift', '~> 2.3.0'
'SwiftHEXColors', '~> 1.0'
```


## Installation

QLSLogger is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "QLSLogger"
```
